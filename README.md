# Discord Bot

A simple Discord bot that shows cryptocurrency prices and system information.

## Features
- Check real-time prices for BTC, ETH, XRP, and DOGE  
- Monitor server status: CPU, RAM, and Disk usage  
- Display system uptime  
- Organized help command: `!com`

## Commands
`!btc` – Bitcoin price  
`!eth` – Ethereum price  
`!xrp` – Ripple price  
`!doge` – Dogecoin price  
`!cpu` – CPU usage  
`!ram` – RAM usage  
`!disk` – Disk usage  
`!info` – System summary  
`!uptime` – Server uptime  
`!com` – Show all commands
